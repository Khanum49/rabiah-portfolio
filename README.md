# Rabiah Khanum | Portfolio

This is my personal portfolio website, built using **Astro** and **Tailwind CSS**, to showcase my professional skills, projects, and technical writing work. 

## Features
- Responsive and modern design
- Sections highlighting technical writing, cloud & networking projects, and coding work
- Docs-as-code style documentation samples
- Easy to update with new projects and content

## Tech Stack
- **Frontend:** Astro, Tailwind CSS
- **Deployment:** Vercel / GitHub Pages (static site)
- **Optional:** JavaScript for interactive components
